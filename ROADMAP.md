# QuizForge AI Roadmap

## v1.0 (MVP)
- Topic input
- MCQ + Short answer quiz
- Difficulty selection
- Scorecard
- Quiz history using Room DB
- JSON-based topic packs

## v1.1
- Bookmarks / Saved questions
- Revision mode
- Timer quiz mode
- UI improvements

## v2.0
- Keyword extraction improvements (RAKE/TF-IDF)
- Topic suggestion module
- Large topic expansion (OS/DBMS/DSA/CN/AIML)

## v3.0
- Optional LLM integration for question generation
- Multilingual support
- Quality scoring and analytics dashboard
