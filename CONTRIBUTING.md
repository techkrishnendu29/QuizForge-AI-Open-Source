# Contributing to QuizForge AI

Thank you for contributing to QuizForge AI.

## Code of Conduct
This project follows the rules in [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md). By contributing, you agree to follow them.

## Prerequisites
- Android Studio (latest stable)
- Kotlin
- Android SDK 24+
- Git

## Setup
1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/techkrishnendu29/QuizForgeAI.git
   ```
3. Open `android-app/` in Android Studio
4. Sync Gradle and run the app

## Branch Naming
- `feature/<feature-name>`
- `fix/<bug-name>`
- `docs/<doc-update>`

## Commit Message Style
Use clear commit messages:
- `Add quiz timer`
- `Fix crash in quiz screen`
- `Update README setup steps`

## Pull Request Rules
- One PR = one feature/fix
- Link PR to an Issue if possible
- Add screenshots for UI changes
- Ensure the project builds successfully before submitting

## Where You Can Contribute
### Beginner Friendly
- Add topic packs in JSON format
- Improve UI design
- Documentation improvements

### Intermediate
- Add features (timer, bookmarks, revision mode)
- Improve Room DB entities/queries
- Improve navigation flow

### Advanced
- Improve NLP pipeline (TF-IDF/RAKE)
- Add CI build checks
- Performance optimization

## Review Process
- Issue response time: 24–48 hrs
- PR review time: 48–72 hrs

Welcome to the community.
